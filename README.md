# 🤖 Sitara ChatBot

A simple command-line chatbot built using Python. This project was created to understand the basics of chatbot development, user input handling, conditional logic, and interactive programming.

The chatbot can respond to common greetings, answer basic questions, tell jokes, and maintain a simple conversation with the user through the terminal.

## Features

* Greets users and responds to common greetings
* Answers basic questions such as its name and purpose
* Responds to positive and negative moods
* Tells a joke when requested
* Provides a help menu with supported commands
* Handles unknown inputs gracefully
* Allows users to exit the conversation using multiple commands

## Technologies Used

* Python 3
* Functions
* Conditional Statements (if-elif-else)
* Loops
* String Manipulation
